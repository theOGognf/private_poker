# pp_bots

Poker bots that adapt to you. Create poker bots with:

```bash
pp_bots --connect $host BOTNAME1 BOTNAME2 BOTNAME3
```

Poker servers are hosted with [pp_server][2] and poker clients can connect
with [pp_client][3].
